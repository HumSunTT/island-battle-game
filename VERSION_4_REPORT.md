## 🎮 新游戏机制详解

### 路径打通机制

**游戏流程**:
```
1. 开局时，玩家只能看到自己角落的岛屿
2. 只能攻击与自己岛屿直接相连的中立岛屿
3. 占领中立岛屿后，解锁该岛屿连接的其他岛屿
4. 逐步打通路径，最终到达其他玩家的岛屿
5. 路径完全打通后，才能攻击其他玩家
```

**策略要点**:
- 🎯 优先打通通往中心岛屿的路径（高收益）
- 🛡️ 保护关键的桥梁岛屿
- ⚔️ 在合适的时机攻击其他玩家
- 💰 平衡扩张和防守

### 岛屿差异化策略

**高血量岛屿**:
- 适合作为防守据点
- 难以被快速攻破
- 建议装备防御道具

**高收益岛屿**:
- 经济优势来源
- 优先扩张目标
- 需要重点保护

### 核心算法

**路径连通性检查（BFS）**:
```typescript
function canReachPlayer(from, to) {
  const queue = [startIsland];
  const visited = new Set();
  
  while (queue.length > 0) {
    const current = queue.shift();
    const connected = getConnectedIslands(current);
    
    for (const island of connected) {
      if (island.owner === currentPlayer) {
        if (island === target) return true;
        queue.push(island);
      }
    }
  }
  
  return false;
}
```
